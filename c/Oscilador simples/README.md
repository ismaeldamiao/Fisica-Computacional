# Baixando
```powershell
wget "https://raw.githubusercontent.com/ismaeldamiao/ismael-damiao-repo/master/c/Oscilador%20simples/Oscilador.c"

```

# Compilando

```powershell
g++ Oscilador.c
```

# Rodando

```powershell
./a.out
```

# Plotando

```powershell
gnuplot plot.gnuplot
```
